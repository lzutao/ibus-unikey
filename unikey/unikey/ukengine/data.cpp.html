25/*
26Instructions on how to add more charset supports
27
28Each charset enumerates all its characters according to a fixed order.
29To understand this order, copy the TCVN3 charset bellow to some application
30and view it with some TCVN3 font.
31
32Steps to add an 1-byte charset:
33
34- Determine the Id for your charset. See "vnconv.h". The Id
35 for your charset is equal to the id of the last 1-byte charset PLUS 1.
36 Then define a constant for that Id (e.g. #define MY_NEW_CHARSET 25)
37- In "vnconv.h": Increase the variable CONV_TOTAL_SINGLE_CHARSETS by 1.
38- Add an entry for your charset to the end of CharsetIdMap (in this file)
39- Add your charset to the end of SingleBytesTable
40- Note that and the end of each charset (after character z) there's a section
41 for the symbols in western charsets (see TCVN3). Just copy
42 this section for your charset, and set zero for each code point
43 that is occupied by your charset (for representing Vietnamese characters).
44
45Steps to add a 2-byte charset:
46- Determine the Id for your charset. See "vnconv.h". The Id
47 for your charset is equal to the id of the last 2-byte charset PLUS 1.
48 Then define a constant for that Id (e.g. #define MY_NEW_CHARSET 44)
49- In "vnconv.h": Increase the variable CONV_TOTAL_DOUBLE_CHARSETS by 1.
50- Add an entry for your charset to the end of CharsetIdMap (in this file)
51- Add your charset to the end of DoubleByteTables
52- Note that and the end of each charset (after character z) there's a section
53 for the symbols in western charsets. Just copy this section from
54 VNI-WIN charset to your charset
55- Double-byte characters are represented as a word in which the
56 low byte is base character, high byte is tone mark (if present).
57*/
